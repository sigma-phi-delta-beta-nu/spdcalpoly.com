# SPD Website Redesign — Image Model Prompt

> One prompt, multiple mockups. Paste into your image model (Midjourney, DAL·E, Nano Banana, Sora-image, etc.). If the model only returns one image, append "— image 1 of 6: the homepage" etc. and run it once per screen.

---

Generate a set of **6 high-fidelity website UI design mockups** for the Sigma Phi Delta fraternity — Beta-Nu Chapter at California Polytechnic State University, San Luis Obispo (an international professional-social fraternity for engineers, founded 1924).

**Shared art direction for ALL 6 images:**
Modern editorial website design, generous whitespace, realistic crisp UI.

**Aspect ratios (important):**
- **Images 1–5 (desktop screens): render at 16:9 landscape**, framed as a clean full-bleed desktop browser screenshot at ~1440×810px. Show the page filling the full viewport width with no side margins / no device chrome beyond the browser frame.
- **Image 6 (mobile screen): render at 9:16 portrait**, framed as an iPhone mockup at ~390×844px (or equivalent 9:16 canvas).

Brand palette is mandated by the fraternity's nationals: **deep crimson red (#8B0000–#A4151E) and near-black charcoal (#141414) as primaries, with gold/brass accent (#C8A951) and a warm ivory/cream background (#FAF7F1)**. Typography: a strong condensed serif or slab display face for headlines, a clean humanist sans for body text. Sparingly use thin gold line-art heraldic/engineering motifs — a stylized castle, drafting compass, retort, winged thunderbolt (these are the fraternity's real symbols). Photography: warm, slightly desaturated college campus / brotherhood photos, with an optional soft crimson duotone treatment. Overall feel: a prestigious professional engineering fraternity — collegiate, trustworthy, a little stately — somewhere between an Ivy engineering school site and a refined modern brand. NOT a generic startup template, NOT teal or blue, NOT neon, NOT clip-arty.
