# Legacy site (archived)

This folder holds the previous **Jekyll + Forty theme** GitHub Pages site that lived at the repository root.

- **`README-jekyll-root.md`** — original root README (theme credits).
- **`_config.yml`**, **`_includes/`**, **`_layouts/`**, **`_sass/`** — Jekyll layout and styles.
- **`*.md`** pages — former homepage and markdown sources.
- **`assets/`** — theme CSS, JS, fonts, and demo images (`pic*.jpg`, banner, etc.). Chapter photo folders **`img_2022`** and **`img_2023`** were kept under repo-root **`assets/images/`** for the Astro site.

To build the old site locally (optional):

```bash
cd .legacy
bundle install
bundle exec jekyll serve
```

The active chapter website is the **Astro** project at the repository root.
